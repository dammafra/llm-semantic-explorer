# 🧭 LLM Semantic Explorer

**Semantic Exploration of LLMs: A Mechanistic Interpretability Approach Through Embedding Visualization**

## Overview

Large Language Models operate on high-dimensional vector spaces, making their decision-making process opaque. This project provides an **interactive 3D visualization tool** that maps the internal latent space of an LLM during text generation, allowing direct observation of how different **prompting techniques** influence the model's trajectory through its embedding space.

The core idea: every token generated by an LLM corresponds to a point in a high-dimensional latent space. By extracting hidden states during auto-regressive inference, reducing their dimensionality, and rendering them in 3D, we can **see** how structured prompts guide the model toward distinct semantic regions — forming measurable clusters, paths, and trajectories.
